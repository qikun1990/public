    MY PROPELLER ADS EXPERIENCE

    I am a first time user of Propeller Ads and I would like share my recent experience with the platform and running some popunder traffic.


    Sign Up
    The sign-up process was short and sweet. Instant approval - gotta like that.


    Add Funds
    Of course, the first thing I want to do is add some funds so I can start running some traffic.

    However, I ran into a small glitch.

    There is a mobile phone verification dialog that appears before you can add funds. I entered my mobile number multiple times, but for some reason, I wasn't receiving the verification code via SMS.

    I don't know if it's just me, but it seems if something technical can go wrong, then it will probably go wrong for me...lol

    So I go in search of the support link which was easy to find on the main dashboard area.

    Thankfully, the support ticket system is integrated into the main platform so no need to register separately. I'm not a big fan of those support systems that require you to register again with another username and password before you can submit a ticket.

    I have submitted a couple of tickets in the last couple of days and the responses have been both prompt and helpful. Big thanks to Mara and Victor from the PA Support Team for their assistance.

    The minimum deposit required is $100 which is very reasonable. I paid with a credit card, but they also accept debit cards and wire transfer.


    Campaign Creation
    OK, my money is in and I'm ready to start creating some campaigns.



    You start by entering campaign details including your tracker URL, frequency capping and geos you want to target. There are some handy preset groups of countries that can be selected also (eg. Europe, Asia, Africa).

    You also enter in the CPM rate you wish to pay and your daily and total budgets for the campaign.

    Propeller Ads currently has a special whereby the minimum CPM for any worldwide traffic is now $0.50 CPM. It would be nice if this minimum CPM was permanent and not just a temporary special

    There is also a handy traffic estimator graphic which helps you decide how much to bid and it will change depending on your targeting options. Being new to the platform, I am not sure how accurate the estimator is, but it seems to be fine as a general guide.



    You also have the standard dayparting/scheduling options and you are able to choose Simple or Advanced Targeting options. The simple options are shown above. The advanced options are shown below.



    The advanced targeting options include connection type, OS, OS version, device, device type, mobile ISP, browser, browser language, proxy, and zone. The zone limitation area is where you can add/whitelist or exclude/blacklist zone ids.

    I like the Add/Exclude switch on each of these options which is coloured blue if you want to add the option and red if you want to exclude it. The red/blue makes it visually easy to see what you have chosen to do.


    Postback URL
    There is a TRACKING menu option on the main dashboard which helps you set up and check your postback URL.



    There are also some step-by-step instructions for those using Voluum.


    Campaign Approvals
    Campaign approvals seem to happen in a timely manner. I have not had an issue with waiting too long to get them approved.


    Dashboard View
    Once my campaigns were approved and running, the Dashboard view allowed me to easily see my campaign stats at a glance.



    The traffic and cost data update every minute which is fantastic. One thing I would like to see is if the Account Balance could also update every minute. At the moment it seems to take much longer.

    If you click on a single campaign in the Dashboard, you see the zone data for that particular campaign.



    From here you can easily Exclude/Blacklist zones that are not performing well. This is a real time-saver because you don't have to go back into the Campaign Settings and manually add each individual zone that you want to blacklist.


    Summary
    There are lots of things to like about the Propeller Ads platform and interface. It's clean, fresh, fast and user-friendly.

    Lots of worldwide traffic is available and conversions are happening for me.

    The only suggestions for improvement I can make right now are to see if you can get the mobile verification working flawlessly when new advertisers want to add funds, and to have the Account Balance update more frequently.

    I look forward to continuing to use Propeller Ads moving forward. Nice work!
